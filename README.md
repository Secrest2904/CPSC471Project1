How to run program: 

    - Open a terminal and run "python3 server.py ###" 
        ### - any port number that is not currently in use
    - Open a second terminal and run "python3 client.py localhost *same port number as server*"
    - In the client terminal you can now run the commands ls, get <filename>, put <filename>, and quit
        - ls will list any files on the server
        - get will download the named file from the server
        - put will upload the named file into the server
        - quit exits the application
